HTML 스타일(Style)

배경색 변경: background-color, 글자색 변경: color, 글자 크기 변경: font-size:250%, 문단 정렬 변경: text-align:center

이미지로 배경화면: background="이미지 경로or주소", HTML 링크(Link), href="/html/main.html"

            target 속성
                1. _blank	           링크로 연결된 문서를 새 창이나 새 탭에서 오픈.
                2. _self	           링크로 연결된 문서를 현재 프레임(frame)에서 오픈. (기본설정)
                3. _parent	           링크로 연결된 문서를 부모 프레임(frame)에서 오픈.
                4. _top                    링크로 연결된 문서를 현재 창의 가장 상위 프레임(frame)에서 오픈.
                5. 프레임(frame) 이름	   링크로 연결된 문서를 지정된 프레임(frame)에서 오픈.

<img src="이미지주소" alt="대체문자열"> alt는 이미지 로딩실패시 나오는 대체 문구

        3. 라디오 버튼(radio)

        4. 체크박스(checkbox)

        5. 파일 선택(file)
